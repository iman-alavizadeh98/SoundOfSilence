# SoundOfSilence
ound of Silence is a rhythm-based combat game set inside a church, where players use musical instruments as weapons. Sound becomes power — melodies damage enemies, and in the boss fight, players defeat a corrupted priest using sharp wordplay, insults, and rhythmic “roasts.” A mix of music mechanics, humor, and stylized combat creates a unique and chaotic gameplay experience.
